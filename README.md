# Batch MP3 to WAV Converter on PyQt5 🎵

## 📝 Описание

**Batch MP3 to WAV Converter** — это десктопное приложение с графическим интерфейсом на PyQt5 для пакетной конвертации MP3 файлов в формат WAV с возможностью настройки параметров аудио.

## 🌟 Особенности

- 🚀 **Пакетная обработка** — конвертируйте множество MP3 файлов за один раз
- 🎚️ **Настройка частоты дискретизации** — от 8 кГц до 192 кГц
- 🔊 **Выбор разрядности** — 16-bit, 24-bit или 32-bit
- 📋 **Удобный интерфейс** — русскоязычный интерфейс с увеличенным шрифтом
- 📊 **Прогресс конвертации** — наглядный прогресс-бар и подробный лог
- 🎯 **Сохранение структуры** — опциональное сохранение структуры подпапок
- ⚡ **Многопоточность** — интерфейс не зависает во время конвертации
- ✅ **Проверка FFmpeg** — автоматическая проверка наличия FFmpeg

## 📖 Использование

- Запустите приложение — откроется главное окно конвертера
- Выберите MP3 файлы — нажмите кнопку "📁 Выбрать MP3 файлы"
- Выберите папку для сохранения — укажите, куда сохранять WAV файлы
- Настройте параметры:
- Частота дискретизации (Гц)
- Разрядность (16/24/32 бит)
    - Нажмите "🚀 Начать конвертацию"
    - Следите за прогрессом — лог покажет все этапы конвертации

### ⚙️ Настройка параметров WAV

- Частота дискретизации
  -  8000 Гц — телефонное качество
  -  22050 Гц — радио качество
  -  44100 Гц — качество Audio CD
  - 48000 Гц — качество DVD/видео
  - 96000 Гц — Hi-Res аудио
  - 192000 Гц — максимальное качество

- Разрядность
  -  16-bit — стандартное качество (как на CD)
  -  24-bit — профессиональное качество (студийная запись)
  -  32-bit — максимальное качество (для профессиональной обработки)
